Задача 1: Поиск сотрудников и их заказов
У вас есть две таблицы:
сотрудники:
id (INT, Primary Key)
имя (VARCHAR)
должность (VARCHAR)
заказы:
номер_заказа (INT, Primary Key)
employee_id (INT, Foreign Key, ссылается на сотрудников)
сумма (DECIMAL)
дата (DATE)
Задание: Напишите запрос, который выведет имя сотрудника, его должность и номер каждого заказа, который он выполнял.
Подсказка: Используйте INNER JOIN для соединения таблиц по employee_id.

Задача 2: Вывод всех сотрудников, включая тех, у кого нет заказов
Задание: Напишите запрос, который выведет список всех сотрудников и информацию о заказах, которые они выполняли. Если у сотрудника нет заказов, в полях заказа должны стоять NULL.
Подсказка: Используйте LEFT JOIN, чтобы также вывести сотрудников без заказов.
Задача 3: Вывод всех заказов и сотрудников, которые их выполняли
Задание: Напишите запрос, который покажет список всех заказов и имена сотрудников, которые их выполняли. Если заказ не привязан к сотруднику, имя сотрудника должно быть NULL.
