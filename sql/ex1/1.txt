В компании "ABC Logistics" создается новая база данных для учета информации о грузах и водителях, которые их перевозят. Ваша задача — создать таблицы и добавить данные, чтобы в будущем было удобно получать нужную информацию.
Условия задачи:
Создайте таблицу Drivers для хранения данных о водителях. Таблица должна содержать следующие столбцы:
driver_id — уникальный идентификатор водителя (целочисленный тип данных, ключевой столбец);
name — имя водителя (тип VARCHAR(50));
license_number — номер водительского удостоверения (тип CHAR(10));
birth_date — дата рождения водителя (тип DATE);
phone_number — телефонный номер водителя (тип VARCHAR).
Создайте таблицу Cargo для хранения информации о грузе. Таблица должна содержать следующие столбцы:
cargo_id — уникальный идентификатор груза (целочисленный тип данных, ключевой столбец);
description — описание груза (тип TEXT);
weight — вес груза в килограммах (тип DECIMAL(6, 2));
load_date — дата загрузки (тип DATE);
destination — место назначения (тип VARCHAR(100)).
Вставьте данные в таблицы:
В таблицу Drivers добавьте информацию о 3-5 водителях.
В таблицу Cargo добавьте информацию о 5-7 грузах.
Составьте запросы для получения информации:
Напишите запрос, который выводит имена водителей и их номера телефонов.
Напишите запрос, который показывает описания и вес грузов, загруженных в текущем году.
Напишите запрос, чтобы найти все грузы, чей вес превышает 1000 кг.
Напишите запрос, чтобы найти всех водителей, чьи номера водительских удостоверений начинаются с цифры "5".
